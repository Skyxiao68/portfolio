# Ben Lai Portfolio

Vanilla HTML/CSS/JS portfolio website.

## Folder Structure
- `index.html` – home page
- `pages/` – projects, about, contact
- `css/` – all styles
- `js/` – all JavaScript 
- `assets/images/` – image assets
- `assets/icons/` – icon assets
- `README.md` – this file

## Technologies
- HTML5
- CSS3
- JavaScript

## Features
- JS‑driven navigation (injected once)
- Light/dark mode with localStorage
- Project filtering (Projects page)
- Contact form validation (real‑time)
- Responsive layout (mobile → desktop)
- Back to top button
